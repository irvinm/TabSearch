## Summary
Find tabs fast by title, URL, or page text. Tab Search temporarily hides non-matching tabs for a clutter-free view, then restores everything when you are done. Includes optional Tree Style Tab support and close-time multi-select.

## Description
**Tab Search helps you find the right tab without tab chaos.**

Instead of scanning dozens (or hundreds) of tabs, type a search term and Tab Search temporarily hides non-matching tabs so you can focus on what matters. When the search ends, your tabs are restored.

**What you can do**
- Search open tabs by URL
- Search open tabs by tab title
- Search loaded page text (content search)
- Use real-time filtering while typing, or manual search
- Find tabs currently playing audio
- Optionally select all matches when the search session ends
- See progress on the toolbar badge while tabs are being processed

**Tree Style Tab support (optional)**
- Works with [Tree Style Tab](https://addons.mozilla.org/en-US/firefox/addon/tree-style-tab/)
- Flattens matched tabs during search for faster scanning
- Restores your original tree state after search
- Optional auto-expand behavior for matched tabs in close-time multi-select workflows

**Privacy and permissions (plain language)**
- This extension uses Firefox tab APIs to hide/show tabs during search
- Hidden tabs are temporary and restored after search
- No external service is required for searching your tabs
- A Firefox permission prompt may appear the first time tab hiding is used

**Important notes**
- Content search works on loaded, regular web pages
- Content search requires at least 3 characters
- Search is case-insensitive substring matching (not fuzzy matching)
